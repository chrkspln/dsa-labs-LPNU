**Програма 3_1.**

Програма обчислює суму ряду цілих парних чисел, доки сума не стане більше 20, використовуючи оператор `while`. 
```c
#include <stdio.h>
#include <stdlib.h> 
#include <iostream> 
int main()
{
int a=0, sum=0, n=0; 
while(sum<20) {
	n+=1;	// Кількість циклів 
	a+=2;	 // Парне число 
	sum+=a; // Сума парних чисел
	printf("n=%d a=%d sum=%d\n",n,a,sum); 
	} 
printf("Loops n=%d\n", n); 
system("pause"); 
return 0;
}
```
